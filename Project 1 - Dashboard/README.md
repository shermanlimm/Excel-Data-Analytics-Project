### Introduction

This data jobs salary dashboard was created for job seekers to investigate salaries for their desired jobs and ensure they are being adequately compensated. 

### Excel Skills Used

The following Excel skills were utilized for analysis:

- Charts
- Formulas and Functions
- Data Validation

### Dataset

The dataset used for this project contains real-world data science job information from 2023. It includes detailed information on:

- Job Titles
- Salaries
- Country
- Skills
