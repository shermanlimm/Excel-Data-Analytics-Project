# Excel Project Data Analytics
Course project focusing on comprehensive Excel skills for data analyst. Consist of 2 projects, [Project 1 - Dashboard](https://github.com/shermanlimm/Excel-Data-Analytics-Project/tree/main/Project%201%20-%20Dashboard) and [Project 2 - Analysis](https://github.com/shermanlimm/Excel-Data-Analytics-Project/tree/main/Project%202%20-%20Analysis)

### Skills covered in these projects:
- Formulas
- Functions
- Charts
- Tables
- Conditional Formatting
- Pivot Tables
- Pivot Charts
- Power Query
- Power Pivot
- DAX
- Data Models
